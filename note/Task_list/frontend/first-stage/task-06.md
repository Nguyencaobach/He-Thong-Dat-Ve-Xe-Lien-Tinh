# Task 06: Dựng Layout dùng chung

## 1. Nội dung công việc
Tạo thư mục `src/components/layout/` và viết các file `site-header.tsx`, `site-footer.tsx`, `admin-sidebar.tsx`.

## 2. Ý nghĩa thực hiện
Chuẩn bị sẵn các mảnh ghép giao diện lớn (Header cho trang công khai, Sidebar cho trang Admin). Các trang cụ thể (Page) sau này chỉ việc "gắn" (import) Layout này vào để dùng chung, đảm bảo tính nhất quán trên toàn hệ thống.
